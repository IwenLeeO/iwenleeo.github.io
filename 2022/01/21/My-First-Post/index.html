3 Hexo

3.1 安装 Hexo

hexo -v 检查是否安装成功

4 创建本地blog

1 在 ubuntu 上新建文件夹 blog
2 输入 hexo init 将 blog 文件夹初始化成一个博客文件夹
3 输入 npm install 安装依赖包。
4 输入 hexo g 生成（generate）网页。
5 由于我们还没创建任何博客，生成的网页会展示 Hexo 里面自带了一个 Hello World 的博客。
6 输入 hexo s 将生成的网页放在了本地服务器（server),浏览器里输入 http://localhost:4000/, 就可以看到刚才的成果了。
7 回到 Git Bash，按 Ctrl+C 结束, 此时再看 http://localhost:4000/ 就是无法访问了

5 新建Bolg

1 /blog, 输入 hexo new “My First Post”在\blog\source_posts 路径下，会有一个 My-First-Post.md 的文件。 编辑这个文件，然后保存。
2 回到 Git Bash，输入 hexo g 输入 hexo s前往 http://localhost:4000/ 查看成果, 按 Ctrl+C 结束
